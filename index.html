Research Focus

We focus on advancing artificial intelligence and machine learning applications in security, particularly through innovative projects that integrate data analytics and cybersecurity frameworks to address real-world threats.

Securing AI Systems

Developing robust ML models resistant to adversarial attacks through novel training techniques and architectural innovations that maintain performance while enhancing security.

PyTorch Adversarial Training Robustness CIFAR-10

AI-Powered Threat Detection

Creating next-generation intrusion detection system machine learning models using deep learning to identify sophisticated cyberattacks.

LSTM Networks Network Traffic Analysis Anomaly Detection TensorFlow

Network Security

Implement new ways of detecting and stopping attacks across networks and devices.

Federated Learning Homomorphic Encryption GDPR Compliance
